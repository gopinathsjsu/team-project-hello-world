# Hello World Hotels

 ## Steps:

### For backend 
1. Inside backend folder, run `pipenv install`
2. Run: `pipenv shell`
3. Go to: src/models/ and make a folder for each of the modules For refrence see hotel folder.


### For frontend
1. Inside frontend folder, run `npm install`
2. `npm start`
